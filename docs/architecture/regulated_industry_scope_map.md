# Regulated Industry Scope Map

Areos scope in this repository is intentionally broader than a single AHU humidity demo.

## Industry packs in scope

- Pharma OSD
- API / chemical manufacturing
- Biotech / biologics
- Advanced food / nutraceutical / specialty chemical / advanced materials manufacturing

## Example assurance chain

Utility event → area → batch/product/material → quality risk → evidence → decision.

## Representative scenarios

- Pharma OSD: humidity excursion, pressure cascade failure, compressed-air dew point excursion
- API / chemical: reactor temperature excursion, nitrogen blanketing pressure drop
- Biotech / biologics: bioreactor pH excursion, WFI conductivity trend, cold-room excursion
- Advanced regulated manufacturing: allergen/cross-contact risk, dry-room humidity excursion

## Product positioning

- Do not sell monitoring; sell proof.
- Existing systems monitor signals; Areos connects signals to validated state, product impact, and audit evidence.
- Human-approved, audit-ready evidence packs.
