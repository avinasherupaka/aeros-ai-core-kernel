# Assurance Engines

Phase 4 in this repo adds practical local implementations of the Areos assurance engines.

## Included engines

- Canonical event normalization
- State-of-control engine
- Event-to-impact engine
- Reliability intelligence
- Evidence graph

## Working now

- Tenant/site/source lineage preserved through events and assessments
- Multi-scenario limit evaluation for humidity, pressure, dew point, reactor temperature, pH, WFI, and cold-room examples
- Practical impact assessment and evidence checklist generation
- Rule/statistics-based recurrence classification
- In-memory evidence graph ready for a future Neptune adapter

## Still scaffolded

- Enterprise persistence backends
- Advanced probabilistic confidence models
- Full workflow authorization/identity controls
- Large-scale event correlation across external enterprise systems
