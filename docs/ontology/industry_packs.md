# Industry Packs

Industry packs in this repo are data/config-driven scenario libraries that tell Areos how to connect source events to impact and evidence expectations.

## Included packs

1. Pharma OSD
2. API / chemical manufacturing
3. Biotech / biologics
4. Advanced food / nutraceutical / specialty chemical / advanced materials

## Each scenario defines

- industry and process area
- likely source systems and protocols
- affected parameters
- sample limits
- possible impacted entities
- quality/safety/compliance risks
- evidence checklist
- suggested personas
